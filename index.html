Pattern Recognition — Lecture Slides

FYS-3012, UiT The Arctic University of Norway — Kristoffer Wickstrøm

Most slides are rendered with jupyter nbconvert --to slides, the same Reveal.js pipeline that RISE uses, so the nested horizontal/vertical slide structure matches what you'd see live in RISE. They're static HTML and served from GitHub Pages, so they never time out.
Decks tagged marimo are exported with marimo export html-wasm and run entirely in your browser via WebAssembly — the Python interpreter, dependencies, and code all live inside the page. Give them a few seconds to spin up on first load.
